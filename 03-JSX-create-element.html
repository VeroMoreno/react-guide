Creando una interfaz de usuario con JSX react sintaxis

Al equipo de React se le ocurrió JSX. JSX nos brinda una sintaxis expresiva para representar nuestra interfaz de usuario, sin perder los beneficios y los poderes de escribir nuestra interfaz de usuario en JavaScript.
