Use KEY al renderizar una lista con React

Es común en React tomar una array y asignar esa array a una array de elementos y representarla directamente en su JSX. Cuando haga eso, es importante que agregue una key al elemento raíz de React de cada elemento en el array para que React pueda rastrear los cambios a lo largo del tiempo y asegurarse de que conserva el estado de cada elemento en el array y sus descendientes.

key={item.id}
