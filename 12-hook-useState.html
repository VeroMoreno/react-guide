Manejar STATE en un componente de react con useState hook

En React, el estado está asociado a los componentes y cuando el estado cambia, el componente se actualiza. Por ejemplo, cuando los datos que un usuario nos proporciona, aquí entra en juego el estado del componente.

Este es el hook desglosado:

const satateArray = React.useState('')

const name = satateArray[0]

const setName = satateArray[1]

const handleChange = event => setName(event.target.value)
