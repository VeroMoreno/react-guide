Uso de los límites de error de React para manejar errores en los componentes de React

Afortunadamente para nosotros, existe una forma sencilla de manejar los errores en nuestra aplicación utilizando un tipo especial de componente llamado Error Boundary . Peeeeero, actualmente no hay forma de crear un componente Límite de error con una función y, en su lugar, debe usar un componente de clase, pero tuvimos otro golpe de suerte porque hay una biblioteca de código abierto excelente que podemos usar llamada react-error-boundary.
