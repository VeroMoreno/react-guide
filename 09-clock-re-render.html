Re-render CLOCK

Normalmente, cada vez que cambia el estado, no tiene que volver a renderizar toda la aplicación, pero quería poner esto para que veas lo que React realmente está haciendo por ti. Cuando crea estos elementos de React, le da eso a ReactDOM.render o activa una re-renderización de un componente, y React va a comparar los elementos que devolvió esta vez con los elementos que devolvió la última vez tiempo.
